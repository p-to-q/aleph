# RFCs

RFCs are optional proposal documents. They are useful when the project needs discussion before a decision is ratified as an ADR or implemented.

Wittgenstein uses research briefs, RFCs, ADRs, exec plans, and code as separate but linked lanes. This template keeps that pattern optional.

## Sample

```text
RFC: introduce a plugin API
Question: what interface should third-party modules implement?
Options: callback registry, manifest-driven plugin, static config only
Decision target: ADR if accepted; no-action note if rejected
```
