# Architecture Decision Records

ADRs are optional. Enable them when the repository has decisions that future contributors should not have to rediscover from chat, PR comments, or old issues.

Use `TEMPLATE.md` and number files as `0001-title.md`.

## Sample

```text
0001-default-license.md
Status: Accepted
Decision: use Apache-2.0 as the default license.
Why: permissive license with explicit patent grant; matches p-to-q practice.
Consequences: package metadata, LICENSE, NOTICE, and README must stay aligned.
```
