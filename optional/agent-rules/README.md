# Agent Rule Templates

This directory stores optional rule files for agentic IDEs and coding assistants.

## Sample

```text
Cursor project:
  copy optional/agent-rules/cursor/project/*.mdc to .cursor/rules/
  copy only matching stack files from optional/agent-rules/cursor/stacks/
  delete unused routes

Non-Cursor project:
  use docs/agent-rules.md and AGENTS.md as the human-readable contract
```
