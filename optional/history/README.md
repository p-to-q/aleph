# History

GitHub remains the source of truth for issues and PRs. This folder stores summaries, audits, and decision maps that help maintainers see patterns without re-reading every thread.

- `issue-pr-ledger.md` — current seed ledger adapted from observed Wittgenstein history.
- `decision-map.md` — durable decisions that should be visible outside GitHub threads.

## Sample

```text
Issue pattern: repeated dependency PR noise
Decision: group patch/minor updates in Dependabot
Durable surface: .github/dependabot.yml and docs/engineering-discipline.md
```
