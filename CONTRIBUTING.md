# Contributing

Start by reading [README.md](README.md), [docs/project-brief.md](docs/project-brief.md),
[docs/architecture.md](docs/architecture.md), and [docs/surfaces.md](docs/surfaces.md).

## Rules

- Make the smallest correct change.
- Label simulated data as simulated.
- Do not claim global optimality for a prompt search result.
- If a public contract changes, update docs or add an ADR.
- Run `npm run lint` before opening a pull request.
