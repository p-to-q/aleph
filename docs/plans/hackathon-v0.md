# Hackathon v0 Execution Plan

## Goal

Ship a credible Aleph Atlas Console that demonstrates the compression path with fixture data.

## Steps

1. Implement core data types.
2. Add sample run fixtures.
3. Build static demo and React scaffold.
4. Add target input, prompt/output, compression slider, Pareto frontier.
5. Add observation panels.
6. Archive design artifacts.
7. Run repo lint and document limitations.

## Definition of done

- Repository zip can be opened and inspected.
- `npm run lint` succeeds without external installs.
- Static demo exists.
- README describes product and roadmap without overclaiming.
