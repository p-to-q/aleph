# Legacy Frontend Archive

This note closes the old `apps/web/` frontend path.

## Status

- `web/` is the only active frontend launch surface.
- `apps/web/` is archived reference material.
- Root npm scripts no longer launch `apps/web/`.

## What remains here

- The old Vite console source and static prototype files.
- Historical receipts that explain how the Hackathon console worked.

## What not to do

- Do not treat `apps/web/` as a live product path.
- Do not wire terminal startup, root workspace, or new product claims to it.
- Do not use its simulated panels as evidence of model internals.
