# Generated Artifacts

This archive preserves design artifacts that shaped Aleph without making them production contracts.

## Kept

- `prototypes/aleph_logo_designs_20.html` — early wordmark/logo combinations.
- `prototypes/aleph_20_uiux_pages.html` — early page-layout directions.
- `prototypes/aleph_uiux_prototypes_v4.html` — later console prototype, if present.
- `conversation-record.md` — compressed record of the decisions made in conversation.
- `template-clone-attempt.log` — environment receipt for the failed direct clone attempt.

## Removed from this iteration

Reference screenshots were removed. They were useful during visual exploration, but keeping them in the repository would make the archive heavier and could confuse future contributors about what is product source, what is reference material, and what is a durable design decision.

Production-facing brand assets live under `apps/web/public/brand/`.
