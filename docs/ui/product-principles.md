# Product Principles

1. The slider is the main narrative, but not the only interaction.
2. Input and output must always be visible.
3. Observability panels are secondary instruments, not decorative charts.
4. Show shortest found, never guaranteed shortest.
5. Leakage is a first-class trust metric.
6. Mock and fixture observations must be labeled.
7. The interface should feel like a calm research instrument, not a prompt marketplace.
