# Interaction Model

## Primary flow

1. Paste target output.
2. Select model, decoding, metric, budget, and mode.
3. Generate compression path.
4. Drag the compression slider or select a Pareto point.
5. Inspect current prompt and model output.
6. Use observation panels to understand fit, loss, attribution, and failures.

## Secondary panels

- Pareto Frontier: selectable map of candidate points.
- Token Loss: per-token/character loss heat text.
- Search Dial: budget and search state.
- Waveform: confidence/loss/stability rhythm across target tokens.
- Attribution Score: prompt token contribution.
- Exposure Vectors: compression risks.
- Eval Suite: reliability checks.
