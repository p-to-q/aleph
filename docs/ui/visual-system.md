# Visual System

Aleph should use a quiet instrument aesthetic:

- warm paper background;
- black/white Aleph symbol;
- serif wordmark for conceptual weight;
- monospace for prompt, token, and run data;
- rounded panels with large whitespace;
- charts that look printable and readable.

Primary palette:

```text
ink      #171615
paper    #efe9dd
panel    rgba(255,255,255,.72)
line     #ddd4c7
amber    #b65f2a
muted    #827b72
```
