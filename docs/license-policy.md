# License Policy

The default license for this template is **Apache-2.0**.

## Where the default appears

- `LICENSE` — full Apache License, Version 2.0 text.
- `NOTICE` — project notice placeholder.
- `package.json` — `"license": "Apache-2.0"`.
- `CONTRIBUTING.md` — contribution license expectation.

## Why this default exists

Apache-2.0 matches the default used by Wittgenstein and several p-to-q repositories. It gives a permissive open-source license with an explicit patent grant.

## Exception policy

Use a different license only when the project surface requires it.

Examples:

```text
Software library or tool       -> Apache-2.0 default
Documentation-only handbook    -> Apache-2.0 or CC-BY-4.0, choose explicitly
Assets, images, media          -> separate asset license may be needed
Private/internal repo          -> keep LICENSE only if publication is plausible
GPL/AGPL dependency boundary   -> add LICENSE_NOTES.md before release
```

## Sample README license line

```markdown
## License

Apache-2.0. See [LICENSE](../LICENSE) and [NOTICE](../NOTICE).
```
