# Fixtures

These sample runs power the Hackathon v0 UI. They are intentionally small and clearly marked as fixture data.

- `sample-run.json` is the default unrestricted demo run.
- `non-leaking-run.json` exercises the same `AlephRun` contract in `non_leaking` mode.
- `manifest.json` indexes available fixture runs without changing the default UI/API fixture path.

Fixture observations are not model evidence. They exist to keep the UI, schema, and contract checks aligned while real adapters are still future work.
